# ROADMAP

## Vision

Build a "Google Maps for Civilizations" that helps users explore long-range
cultural evolution through time, geography, layers, and influence.

## Phases

1. Phase 1: Core data model and seed dataset
   - Define normalized tradition schema
   - Track uncertainty and evidence quality
   - Seed initial South Asian tradition records
2. Phase 2: Timeline river visualization
   - Interactive timeline lanes
   - Basic query-driven filtering
   - Selection detail panel
   - Synchronological chart mode for parallel-history reading
3. Phase 3: Geographic map
   - Region geometry model
   - Diffusion paths and zone overlays
   - Map-time synchronization
4. Phase 4: Influence network graph
   - Directed influence edges
   - Edge confidence and evidence references
   - Node clustering by layer and region
5. Phase 5: Civilizational stack
   - Layered strata model
   - Coexistence and overlap visualization
   - Layer composition analysis
6. Phase 6: Interactive query engine
   - Structured query grammar
   - Natural-language translation layer
   - Explainable filter logic
7. Phase 7: Optimization and UX refinement
   - Performance for large datasets
   - Narrative scroll scenes
   - Accessibility, onboarding, and export

## Immediate Next Steps

1. Replace seed assumptions with source-cited records.
2. Introduce explicit influence edge objects instead of embedded ID arrays.
3. Add region entities and prepare map-ready geometry references.
4. Migrate UI into a component framework once the domain model stabilizes.
