# Civilization Engine

Civilization Engine is the first baseline for an interactive application that
visualizes civilizational and religious evolution across time. This initial
iteration focuses on:

- a documented domain model for traditions
- a seed dataset with explicit uncertainty fields
- multiple interactive visualizations:
- a timeline river
- a synchronological chart
- baseline planning documents for long-term iterative development

## Run

```bash
npm run dev
```

Then open [http://localhost:4173](http://localhost:4173).

## Test

```bash
npm test
```

## Current Scope

This seed dataset is intentionally small and uses approximate chronology for
interface prototyping. It is not yet a source-cited historical corpus.
