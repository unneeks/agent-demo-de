# KNOWN_LIMITATIONS

1. The current dataset is a prototype and not yet source-cited.
2. Several records are umbrella categories, not fine-grained traditions.
3. Geography is text-only; no map geometry or diffusion paths exist yet.
4. Influence relationships are simplified and untyped.
5. The query engine uses a very small rule-based parser.
6. The visualization is currently a timeline only; map, stack, and graph views are pending.
